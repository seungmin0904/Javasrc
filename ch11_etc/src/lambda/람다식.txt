람다식 : 익명 구현 객체를 식으로 표현

람다식으로 사용할 수 있는 타겟타입(인터페이스)
-추상메소드를 하나만 가지고 있어야 함 
- @FunctionalInterface (추상메소드가 하나인지 확인) 사용으로 컴파일 시 오류 검출


표준 API 의 함수적 인터페이스
1) Consumer : 매개값 있고 리턴값 없음 (소비)
2) Supplier : 매개값 없고 리턴값 있음 (반환)
3) Function : 매개값, 리턴값 있음 / 주로 매개값을 리턴값으로 매핑
4) Operator : 매개값, 리턴값 있음 / 주로 매개값을 연산하고 결과를 리턴
5) Predicate : 매개값 있고 리턴값 boolean / 매개값을 조사해서 t/f 리턴
